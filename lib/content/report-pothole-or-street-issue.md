## 1. Report an issue {#report-an-issue}

Report potholes, broken pavement, and damaged signs online.
Provide the exact location and a brief description of the issue.

:::custom-button{href="/streets/report"}
Report a Street Issue
:::

## 2. What to include {#what-to-include}

Include the street name and nearest cross streets.
If it is safe, attach a photo so crews can locate and assess the problem quickly.

:::custom-alert{type="error" title="Emergency Hazards"}
If a sinkhole, downed wire, or flooding poses an immediate physical threat, call 911 immediately.
:::

## 3. Track repair status {#track-repair-status}

You will receive a reference number to track your request.
Response times depend on severity, season, and crew workload.

:::custom-card{title="Check Status" buttonText="Track Request Status" buttonLink="/streets/track"}
View the progress of your street repair request online.
:::

:::custom-dropdown{title="Why is it taking so long to fix?"}
Critical hazards are prioritized. Weather events, like snow, can severely delay standard paving operations.
:::
